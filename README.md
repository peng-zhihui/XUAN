## 文件说明

`Hardware`里面是控制器的PCB文件，基于ESP32，搭载MPU6050，通过CAN总线连接驱动器。

`Structure`里面是车身结构设计文件，step是Fusion导出的可能有些BUG，建议直接下载这个Fusion360的工程在软件里打开：https://a360.co/2TOtZRd
